I have implemented the `tooltips` component into markdown which is ready to use.

## Preview

<div className="w-full">
  <Tooltip tip="this is tooltips! DocuBook on version 1.0.5">
    <span style={{ textDecoration: "underline dotted", cursor: "pointer" }}>
      Hover over me
    </span>
  </Tooltip>
  <span> and this is some regular text.</span>
</div>

## Props

| Prop   | Type     | Default | Description                   |
| ------ | -------- | ------- | ----------------------------- |
| `tip`  | `string` | `null`  | The value of show of tooltip. |
| `span` | `string` | `""`    | The text style for tip        |
